# deployment

## Tools

### GitHub Issues Listing Tool

A Python script to list and filter GitHub issues from the command line.

**Quick Start:**
```bash
python list_issues.py --owner ASE-G1 --repo deployment
```

**Documentation:**
- [Full Documentation](ISSUES_TOOL_README.md) - Complete guide with all parameters and examples
- [Quick Reference](ISSUES_QUICK_REFERENCE.md) - Common commands and use cases
- [Parameters Summary](PARAMETERS_SUMMARY.md) - Detailed list of all parameters

**Required Parameters:**
- `--owner`: Repository owner (e.g., ASE-G1)
- `--repo`: Repository name (e.g., deployment)

**Optional Parameters:** state, labels, assignee, creator, since, sort, direction, per-page, page, json

See the documentation files for detailed usage examples.
